Toolbar icon

The Inspekt icon in your toolbar changes per tab to describe what's available:

  • Greyscale — page has no Inspekt instrumentation; the inspector won't find any sources here.
  • OFF Page is instrumented but the inspector is disabled. Click the icon to enable.
  • ON Active — file path only (no snippet source available).
  • DEV Active — snippets via the local dev server.
  • MAP Active — snippets via source maps (opt-in fallback).
  • AI Active — agent connected via MCP. Grabs route to your editor's agent.

Source snippet

When you click an element, Inspekt fetches the surrounding lines from your dev server's /__inspekt/snippet endpoint and renders them in the popover. Works automatically when @inspekt/vite is running.

When off, the popover shows a "Show source ▾" toggle and only fetches on first expand.
Lines above and below the target line (max 30).

Source map fallback (advanced)

Inspekt can fetch source-map files from your app's bundle URLs when the local dev server isn't running. Useful for inspecting staging/production sites or extension pages.

When enabled, your browser will make additional network requests to download .map files. These requests appear in your DevTools Network tab and in corporate proxy/network logs. Map files can be several MB and are cached for the browser session.

Disable when on shared / monitored networks, or when inspecting sites where source maps shouldn't be requested.
